web3j-homebrew
==============

Homebrew Tap for web3j command line tools.

Getting started
---------------

Add the tap, and install:

.. code-block:: bash

   brew tap web3j/web3j
   brew install web3j

   
Run the web3j executable:

.. code-block:: bash

   $ web3j
